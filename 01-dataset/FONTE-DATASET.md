# Dataset - Gastos Parlamentares (CEAP)

## Fonte
- Kaggle: https://www.kaggle.com/datasets/joniarroba/how-brazilian-politicians-use-their-quota
- Arquivo: Gastos-Quota-Parlamentar.csv
- Tamanho: 403 KB
- Download realizado em: 02/03/2026

## Descrição
Dataset contendo os gastos da Cota para Exercício da 
Atividade Parlamentar (CEAP) dos Deputados Federais do Brasil.

## Colunas Principais
- congressperson_name: Nome do deputado
- party: Partido político
- state: Estado
- term: Mandato
- subquota_description: Tipo de gasto
- supplier: Fornecedor
- net_value: Valor do gasto
- year/month: Período
