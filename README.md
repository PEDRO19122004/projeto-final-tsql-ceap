# Projeto Final T-SQL — Análise de Gastos Parlamentares (CEAP)

## Descrição
Análise dos gastos da Cota para Exercício da Atividade Parlamentar
dos Deputados Federais do Brasil, utilizando T-SQL no SQL Server.

## Dataset
- Fonte: Câmara dos Deputados — dadosabertos.camara.leg.br
- Volume: ~2 milhões de registros (2009-2024)
- Formato: CSV
